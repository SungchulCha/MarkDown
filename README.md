## <br>3. Emphasis
  - ### *italic*
     ```
        *italic*
        _italic_
        <i>italic</i>
     ```
  -  ### **Bold**
     ```
        **Bold**
        __Bold__
        <b>Bold</b>
     ```
  - ### Cancel Line
     ```
        ~~Cancelline~~
        <del>Canceline</del>
     ```
  - ### Under Line
     ```
        <u>Underline</u>
     ```
